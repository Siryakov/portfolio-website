Readme файл
Этот репозиторий содержит код для создания адаптивного веб-сайта портфолио. Ниже приведены основные компоненты и настройки, используемые в коде.

Стилизация
Шрифты:
Шрифт для основного текста: Mulish
Шрифт для заголовков: Merriweather

Цветовая схема
Цвет фона шапки: #222222
Цвет текста и ссылок: #fff
Цвет фона некоторых секций: #333333

Структура HTML
header: шапка сайта с логотипом и ссылками на социальные сети
banner: баннерная секция с основным текстом и кнопкой
Citation: секция с цитатой
gallery: секция с работами, включая анимированные изображения
services: секция с услугами и описаниями
footer: подвал сайта с информацией о разработчике и ссылками на контакты

Адаптивный дизайн
Сайт разработан с использованием медиа-запросов для обеспечения адаптивности на различных устройствах!!!
При ширине экрана менее 1200px галерея работ отображается в центре, а заголовки становятся по центру.
При ширине экрана менее 900px услуги отображаются в две колонки вместо трех, и заголовки становятся по центру.
Для более подробного изучения кода и его использования рекомендуется ознакомиться с HTML и CSS файлами, предоставленными в репозитории.
